- [Typescript](https://www.typescriptlang.org/)

- [Uuid](https://www.npmjs.com/package/uuid)
- [Swagger](https://swagger.io/)

### 💻 Installation

1. Clone the repo
   ```sh
   git clone https://github.com/saboyagustavo/products-API.git
   ```
2. Install the packages
   ```sh
   cd products-API
   yarn
   ```

### 👩‍💻 Run locally

1. Start the server
   ```sh
   yarn start
   ```
